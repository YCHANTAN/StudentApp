## 📱 App Structure & Navigation
The application is organized into five primary navigation pillars:

1. **Home (Dashboard):** Summary of balances, grades, notifications, and QR identifier.
2. **Academic:** Management of programs, courses, prospectus downloads, and evaluations.
3. **Finance:** Viewing balances, downloading payment slips, and transaction history.
4. **Services:** Document requests (TOR, COE), Library systems, and SAO complaints/requests.
5. **Profile:** Personal info management, security settings (2FA/OTP), and emergency contacts.

---

## 🛠️ Key Features
* **Document Tracker:** Real-time status updates (Processing, Accepted, Ready to Go).
* **Library System:** Check book availability, borrow/return logs, and history.
* **Downloads:** Offline access to Study Loads, QR Codes, and Academic Records.
* **Security:** Integrated 2FA/OTP verification and encrypted profile management.
